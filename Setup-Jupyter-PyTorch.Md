Jupyter Notebook with PyTorch

✅ Prerequisites
- VSCode installed (Windows version).
- Python installed (preferably via Anaconda or directly from python.org).
- PyTorch installed (via pip or conda).
- VSCode extensions:
- Python (by Microsoft)
- Jupyter (by Microsoft)


Step-by-Step Setup
- Install Python & Jupyter
- If using pip:
pip install notebook jupyterlab
- If using conda:
conda install -c conda-forge notebook jupyterlab
- Install PyTorch
- Go to PyTorch’s official site and copy the install command for your setup (CPU or GPU).
- Example (CPU only):
pip install torch torchvision torchaudio
- Open VSCode and Install Extensions
- Install Python and Jupyter extensions from the VSCode marketplace.
- Restart VSCode after installation.
- Create a Jupyter Notebook in VSCode
- Open VSCode → File > New File → Save as example.ipynb.
- VSCode will recognize it as a Jupyter Notebook and open an interactive notebook editor.
- Select Python Environment
- At the top right of the notebook, select the Python interpreter that has PyTorch installed.
- If you installed via Anaconda, choose the corresponding conda environment.
